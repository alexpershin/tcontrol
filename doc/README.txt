"tcontrol" - is the project to use with open-source and/or free technology and/or software.

It is intended to check the conditions of a temperature sensors remotely. 

All we need to do are:
1. Create the SQL-script to create a new DB;
2. Create Java-backend app to work with created DB;
3. Create an iOS app to work with this base;
4. Create an Android app to work with this base;
5. Comply all together and test it all;
6. Increase functionality in the future.
